# WDM Simulation

## Project Description
Wavelength Division Multiplexing (WDM) is a technology that uses different wavelengths (or channels) to transmit data over the same fiber optic cable simultaneously. This project simulates WDM systems and demonstrates how multiple data streams can effectively be transmitted.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/sarveshvengurlekar/WDM_Simulation.git
   ```
2. Navigate to the project directory:
   ```
   cd WDM_Simulation
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
Run the simulation by executing the following command in your terminal:
```python
python main.py
```

## Features
- Simulates multiple data streams using different wavelengths.
- Visualizes the WDM system's performance and efficiency.
- Easy configuration of wavelengths and data rates.
